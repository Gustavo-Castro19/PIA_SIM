# ONAC - Lista PIA

Projeto acadêmico desenvolvido para o ONAC.

## Banco de Dados

Estrutura criada em PostgreSQL para gerenciamento da Lista de Pessoas de Interesse Antifraude (PIA).

### Tabelas

- usuario
- incidente
- incidente_analise
- pia
- pia_historico

### Recursos

- Relacionamentos entre tabelas
- Chaves primárias e estrangeiras
- Índices para performance
- Triggers de auditoria
- Views para consultas seguras
- Dados iniciais (seed)

### Tecnologias

- PostgreSQL
- pgAdmin 4
